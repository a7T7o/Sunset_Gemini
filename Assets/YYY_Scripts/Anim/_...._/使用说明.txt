工具动画系统 - 使用说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【工具1：图层动画设置工具】
功能：批量生成工具动画，同步Hand和Axe的Pivot

使用步骤：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开工具：Tools → 图层动画设置工具

2. 配置输入：
   原动画Clip: 拖入原动画（如 Slice_Down_Clip）
   Hand文件夹: 拖入 Down/Hand
   Axe文件夹: 拖入 Down/Axe
   
3. 配置输出：
   输出文件夹: Assets/Animations/Tools（或自定义）
   
4. 配置设置：
   动画名称: Slice_Down_Clip
   工具类型: 0（0=斧头, 1=稿子...）
   动画总帧数: 100
   最后一帧: 80
   
5. 点击【批量创建工具动画】

6. 结果：
   ✅ Hand的Pivot已同步（不生成动画）
   ✅ 生成Axe动画：
      Slice_Down_Clip_0_0.anim（木斧）
      Slice_Down_Clip_0_1.anim（石斧）
      Slice_Down_Clip_0_2.anim（磨石斧）
      ...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【工具2：工具动画控制器生成】
功能：扫描动画文件夹，自动生成Animator Controller

使用步骤：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开工具：Tools → 工具动画控制器生成

2. 配置输入：
   工具动画文件夹: 拖入包含所有动画的文件夹
   
3. 配置输出：
   输出文件夹: Assets/Animations/Tools
   
4. 点击【生成控制器】

5. 结果：
   生成多个Controller：
   Tool_0_0.controller（类型0品质0）
   Tool_0_1.controller（类型0品质1）
   ...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【完整工作流程】

处理Down方向：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 准备资源（已完成Sprite Editor切分）：
   Down/Hand/Slice_Down-Sheet
   Down/Axe/Axe_0, Axe_1...Axe_5

2. 运行"图层动画设置工具"：
   → 同步Hand和Axe的Pivot
   → 生成6个Axe动画

3. 重复处理Side和Up方向

4. 所有方向完成后，运行"工具动画控制器生成"：
   → 自动扫描所有动画
   → 生成所有Controller

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【注意事项】

1. 资源准备：
   ✓ 所有sprite必须在Sprite Editor中切分
   ✓ Sprite Mode必须为Multiple
   ✓ 确保sprite展开后有多个子项

2. 命名规则：
   Axe文件：Axe_0, Axe_1, Axe_2...
   动画输出：{名称}_{类型}_{品质}.anim
   
3. 如果遇到sprite无法显示：
   → 打开Sprite Editor
   → 直接点击【应用】
   → 重新运行工具
